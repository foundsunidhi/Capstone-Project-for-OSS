## Project Description
This project presents an analysis of Python as an open-source software. It covers the origin of Python, its licensing model, ethical aspects, ecosystem, and comparison with proprietary software.  

Along with theoretical understanding, the project also includes practical implementation through shell scripting. Five scripts have been developed to demonstrate basic Linux commands, automation, and problem-solving using Bash scripting.

The main objective of this project is to understand how open-source software works and how it contributes to modern software development.

---

## Scripts Overview

### 🔹 Script 1: System Identity Report
Displays system-related information such as kernel version, username, system date, and basic system details.

### 🔹 Script 2: FOSS Package Inspector
Checks whether Python is installed on the system and displays its version along with a short description.

### 🔹 Script 3: Disk and Permission Auditor
Analyzes selected directories and displays their permissions and disk usage.

### 🔹 Script 4: Log File Analyzer
Reads a log file, counts occurrences of a keyword (like "error"), and displays matching lines.

### 🔹 Script 5: Open Source Manifesto Generator
Generates a personalized open-source manifesto based on user input and saves it into a text file.
